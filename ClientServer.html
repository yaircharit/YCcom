Client side developed in C++, Server side developed in Python.
All users need to register to the server- all users are stored in an SQL database.
Use AES encryption to establish symmetric RSA encryption keys with other users.
Once a connection was established encrypted messages can be sent between users, a message can be either some text and even a file.
All (encrypted) messages are stored in an SQL table ready to be fetched.
All messages are deleted after fetching them.
